Kebijakan Privasi

Aplikasi milik Organisasi: Vision Sukses Makmur

1. Pendahuluan

Kebijakan Privasi ini menjelaskan cara Vision Sukses Makmur mengumpulkan, menggunakan, menyimpan, dan melindungi data pengguna saat menggunakan aplikasi kami.

2. Informasi yang Dikumpulkan

Aplikasi ini tidak mengumpulkan data pribadi pengguna untuk dikirim, disimpan, atau diproses di server mana pun.

3. Tujuan Penggunaan Data

Seluruh data aplikasi digunakan hanya untuk menjalankan fungsi aplikasi di perangkat pengguna (local device processing) tanpa upload ke server eksternal.

4. Penyimpanan dan Perlindungan Data

Penyimpanan data dilakukan secara lokal di perangkat pengguna. Kami tidak menyediakan mekanisme penyimpanan cloud maupun sinkronisasi ke server pihak kami.

5. Pembagian Data kepada Pihak Ketiga

Karena aplikasi tidak mengumpulkan dan tidak mengunggah data pengguna ke server, tidak ada data pengguna yang dibagikan, dijual, atau ditransfer ke pihak ketiga.

6. Hak Pengguna

Pengguna memiliki kendali penuh atas data lokal pada perangkatnya, termasuk untuk:

  • Menghapus data aplikasi melalui pengaturan perangkat.
  • Mencopot pemasangan aplikasi kapan saja.

7. Pengguna Usia 13 Tahun ke Atas

Aplikasi ini ditujukan untuk pengguna berusia 13 tahun ke atas. Kami tidak secara sengaja mengumpulkan data pribadi dari anak di bawah usia 13 tahun.

Jika Anda adalah orang tua atau wali dan mengetahui adanya data pribadi anak di bawah 13 tahun yang terkait dengan aplikasi ini, silakan hubungi kami agar dapat ditinjau dan ditangani sesuai ketentuan yang berlaku.

8. Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu. Perubahan akan berlaku sejak tanggal pembaruan yang tercantum di halaman ini.
